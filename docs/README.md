# 기능 구현 목록

> Controller
- 메서드 흐름 제어
- 게임 시작 기능
- 게임 초기화 기능
- 게임 진행
- 다리를 건너는 기능
- 다시 시작 여부를 묻는 기능
> Service
- 게임 시작 메세지 출력 기능
- 게임 초기화 기능
- 다리 만드는 기능
- 유저 키입력 리스트 초기화 기능
- 다리를 건너는 기능
- 다리 마지막까지 도달했는지 boolean값 반환
- 다시 게임을 시도할지 물어보는 기능
- 에러메세지 출력 기능
> BridgeGame
- 다리 생성 기능
- Bridge 객체 유저가 입력할 키 리스트 초기화
- 유저가 입력한 키 리스트 추가 기능
- 다리 끝가지 돌아간 경우 boolean값 반환 기능
> BridgeMaker
- 다리를 생성하는 기능
> BridgeRandomNumberGenerator
- 0과 1을 랜덤으로 반환하는 기능
> InputView
- 다리의 길이를 입력받는 기능
    - 공백을 입력한 경우 예외처리
    - 숫자가 아닐경우 예외처리
    - 2자리를 넘어선 문자열인 경우 예외처리
    - 3이상 20이하의 숫자가 아닐경우 예외처리
- 사용자가 이동할 칸을 입력받는 기능
  - `U`와 `D`가 아닌 입력값의 경우 예외처리
  - 공백을 입력한 경우 예외처리
- 사용자가 게임을 다시 시도할지 종료할지 여부를 입력받는 기능
  - `R`과 `Q`가 아닌 입력값의 경우 예외처리
  - 공백을 입력한 경우 예외처리
> OutputView
- 게임 시작 메세지 출력 기능
- 다리의 길이를 입력하라는 메세지 추가
- 현재 다리의 상태를 정해진 출력하는 기능
- Bridge 맵 출력
- Bridge 맵 한줄 생성 기능
- 에러메세지 출력 기능
- 게임을 다시 시도할지 여부를 입력하라는 메세지 출력 기능
